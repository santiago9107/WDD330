Week 4 Notes:

- It is better to use square bracket notation in order to reference the form element and avoid mistakes referencing other parts of the code that we want not.

- form.elements returns an HTML collection of all elements contained in the form.

- select menus: for drop-down lists of options

- The placeholder attribute will insert the value provided in the input field until the user enters some text.

- simply button creates a clickable button that can have an event listener attached to it

- menuitem and li tags to create a dropdown menu when it’s clicked on

- JavaScript validation should be used to enhance the user experience when filling in a form by giving feedback about any errors before it’s submitted.

- A submit button can be disable by added the disabled attribute to the input element

- An alternative way to create objects is to use a constructor function: This is a function that defines the properties and methods of an object.
