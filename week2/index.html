Week 2 Notes:

- undefined, NaN, Infinity are not reserved words but should have been.

- Primitive data types: string, symbol, number, boolean, undefined, null.

- Any value that is not a primitive data is an object.

- It is possible to find the type of a value using the operator typeof.

- JavaScript uses the keywords const and let to declare variables.

- const is used when the variable will not be reassigned to another value.

- let is used if the variable might be reassigned later in the program.

- Use global variables as less as possible.

- Using let or const to declare variables will ensure they are block scoped, and it is good practice.

- Variables that start with an underscore generally refer to private properties and methods

- When const is used, it is a good habit to finish the line with a semicolon.

- For last occurrence of a character use the lastIndexOf() method.

- Use includes() to know if certain character is included in a setring.

- try to be very explicit about the types of values you are working with.

- Using double negation (!!) it is possible to find out if a value is truthy or falsy

-The delete operator will remove an item from an array. However, the item will be considered as undefined.

- Taking the values out of an array and presenting them as individual values is called destructing an array.

- Use pop() method to remove the last item of an array. shift() method removes the first item.
